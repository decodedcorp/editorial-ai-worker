## Decisions Made

1. **hatchling over uv_build** - uv_build (auto-generated by `uv init --lib`) expected `editorial_ai_worker` package name matching the project name. Since the desired package name is `editorial_ai`, switched to hatchling which supports explicit `packages` config.
2. **Python 3.12+ type syntax** - Used `str | None` instead of `Optional[str]` since target is Python 3.12 and ruff UP rules prefer modern syntax.
